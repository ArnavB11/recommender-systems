# Multi-Objective Recommendation System (MSRS)

This repository implements and evaluates multi-objective recommendation list optimization using PyTorch (for Neural Collaborative Filtering) and `pymoo` (for evolutionary search with NSGA-II). The system balances three conflicting stakeholder objectives:
1. **Personalized Genre Novelty (DOPM):** Balancing accuracy and diversity.
2. **BERT Semantic Serendipity:** Finding unexpected yet highly relevant recommendations using IMDb review text embeddings.
3. **Provider Exposure Fairness:** Distributing exposure across long-tail creators.

---

## Algorithms and Models

### NeuMF Base Recommender
A neural collaborative filtering (NCF) model combining Generalized Matrix Factorization (GMF) and Multi-Layer Perceptron (MLP) layers in PyTorch to output rating predictions in `[0, 1]`.

### FAS-MOEA (Baseline)
Implements the baseline multi-objective evolutionary algorithm focusing on accuracy, fairness, and serendipity as outlined in Khaitan & Shrivastava (2026).

### MSRS (Standard Multi-Objective Recommendation System)
An optimized NSGA-II approach featuring:
*   **PSNR Sweet-spot Filter:** Variant lists generated by weighted roulette are checked. Only lists whose NCF score variance falls in the sweet spot of the original list are kept.
*   **Weighted Roulette Seeding:** Population is seeded using top NCF rating predictions as probabilities.
*   **Standard Swap Mutation:** Maintains stability of lists while optimizing.
